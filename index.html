Premium AutoPay

Automatic payment confirmation without asking sellers for IMAP settings.

Premium servers can forward only PayPal or Cash App payment notification emails to a unique SMOKNetworks forwarding address. Store Bot reads the forwarded notifications, matches the order code and amount, then marks the order paid automatically.

Privacy notice

Forward only payment notifications.

Forwarding AutoPay only needs PayPal or Cash App payment notification emails. Store Bot uses forwarded email data such as provider, amount, currency, order code, server forwarding alias, and payment status to match pending orders. Server owners should not forward their full inbox.

Questions

Quick answers before you install.

What can I sell with Store Bot?

VIP access, license keys, serial keys, configs, ZIP downloads, setup services, game server perks, and other digital products.

Do you offer bot hosting?

Yes. Bot hosting is available separately through the SMOKNetworks Discord support server. Store Bot itself focuses on storefront panels, order management, AutoPay, logs, staff roles, discounts, and gift cards.

Does it process payments directly?

No. Store Bot does not act as a payment processor. Free servers can handle payments manually. Premium servers can use forwarding AutoPay to detect forwarded PayPal or Cash App payment notifications and mark matching orders paid.

Do server owners have to enter IMAP settings?

No. Forwarding AutoPay lets premium owners forward only PayPal or Cash App payment notification emails to a unique Store Bot forwarding address. The older IMAP setup can remain available as an optional mode for people who prefer direct inbox scanning.

Can staff manage orders?

Yes. Server owners can configure staff roles and a logs channel so trusted team members can manage products and orders.

Is the homepage store preview real?

No. The homepage preview is a demo only. The product names, prices, and order code are placeholders to show how Store Bot can look.

Is it mobile friendly?

The bot is managed inside Discord, and this website is built mobile-first so users can read, invite, and get support from any device.
