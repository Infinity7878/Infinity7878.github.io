Setup Flow

Simple workflow for your CS2 server.

  1. Install the CounterStrikeSharp plugins on your CS2 server.
  2. Generate VIP redeem codes from the CS2 server console.
  3. Add those codes to the Discord bot stock.
  4. Post the Discord store panel.
  5. Players buy, then receive a redeem code, serial key, or file download automatically.
