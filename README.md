# SMOKNetwork Plugins GitHub Pages Site

Static landing page for SMOKNetwork CS2 plugins and Discord VIP store bot.

## Files

```txt
index.html
styles.css
script.js
```

## How to publish with GitHub Pages

1. Create a new GitHub repo.
2. Upload these files to the repo root.
3. Go to:

```txt
Repo → Settings → Pages
```

4. Under **Build and deployment**, set:

```txt
Source: Deploy from a branch
Branch: main
Folder: /root
```

5. Save.

Your site will be available at:

```txt
https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/
```

## What to edit

In `index.html`, replace links if needed:

```txt
https://discord.gg/zB7NgPBzBA
https://github.com/Infinity7878/SMOKNet-VIP-Discord-Bot
```

Update prices in the pricing section if your product prices change.
