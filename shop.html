How pages are created

The website does not let random users create shops.

A server gets a public shop only when Store Bot has exported that server's approved shop data. The frontend only reads the published JSON files.

Store Bot is invited to the seller's Discord server. The server owner enables public Shop/Reputation pages. The bot exports products, policies, order counts, and verified reviews. The website searches and renders the exported data.

Security note

Reviews must come from Store Bot orders.

There is no public “add review” button. If a review is shown here, it should come from the bot after a real order was completed and reviewed inside Discord.
