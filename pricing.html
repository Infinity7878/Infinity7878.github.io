Which plan should I use?

Use Free to test the panel, products, and order flow. Use Premium when you are actively selling and want more product slots, cleaner branding, AutoPay forwarding, discounts, gift cards, staff roles, and logs. Premium is $3/month after the $1 first-month offer for first-time users.

How do I upgrade?

Join the support server and ask for Premium. Mention the first-time user offer if this is your first Premium server. Premium is handled through Discord so the right server can be upgraded.
