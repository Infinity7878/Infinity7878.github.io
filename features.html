Features

Store tools that fit how Discord servers already work.

Store Bot is built around panels, commands, staff logs, and order statuses so you can sell without sending every customer through a separate website.

01

Product panels

Publish a storefront panel in your server. Customers choose a product and start the order flow without needing a separate checkout page.

02

Order statuses

Keep track of pending, paid, delivered, cancelled, and completed orders. Staff can see what needs attention instead of searching old messages.

03

Delivery text

Attach delivery notes, license details, setup instructions, or access information to products so fulfillment is consistent.

04

Staff roles

Give trusted members management access without giving the whole server control over the store.

05

Logs

Send order updates and store actions to a dedicated channel so owners and staff have a clear paper trail.

06

Premium tools

Unlock more product slots, AutoPay forwarding, discounts, gift cards, custom branding, and a cleaner customer experience.

Built for digital products and services

Store Bot works best for products that can be delivered through text, files, roles, access codes, support handoffs, or staff-managed service queues.

  • License keys, serial keys, configs, and ZIP downloads
  • VIP access, private roles, server perks, and gaming community add-ons
  • Plugin setup, Discord services, software services, and custom work

Owner-friendly controls

The setup commands are meant for server owners who want a store running quickly without building their own website or custom checkout system first.
